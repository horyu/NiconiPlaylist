# NiconiPlaylist TODO

本書は NiconiPlaylist の将来拡張、改善案、未実装の検討項目を整理する。

現行仕様の正本は [spec.md](./spec.md) を参照する。

## 1. オプションページ

- プレイリストに作成日時を付与する
- プレイリストの再生を開始する
- プレイリストがポップアップに表示されないようにする
- 動画一覧の検索や絞り込みを行う

## 2. プレイリスト生成

- 再生終了時に完走日時を保存する
- 公式の検索結果、マイリスト、投稿者ページから、現在の表示順をもとにプレイリストを生成する

## 4. 外部連携

- NiconiCompare の CSV/TSV からの取り込み
- GitHub Pages 上のプレイリスト生成 UI
