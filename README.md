# NiconiPlaylist

ニコニコ動画に独自のプレイリスト機能を提供するブラウザ拡張機能です。

## クイックスタート

```powershell
mise trust
mise install
bun install
bun run vp config
```

`bun run vp config` は、このプロジェクトで使用される Vite+ の Git フックを設定します。フックをインストールしたくない場合は、このコマンドをスキップしてください。

## Documentation

設計・仕様書の正本は `project-docs/` に置く。  
`docs/` は GitHub Pages 公開用ディレクトリとして扱う。
